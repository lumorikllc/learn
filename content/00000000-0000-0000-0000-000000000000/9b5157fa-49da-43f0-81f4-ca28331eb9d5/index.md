### üåê Module 3: Networking Fundamentals

Cover the basic principles of networking, including the OSI and TCP/IP models, IP addressing, and common protocols. Learn to design and document network topologies and connect devices using proper cabling. This module lays the groundwork for more advanced Network+ and CCNA topics.

**Topics covered:**

- Explain OSI vs. TCP/IP models and map protocols to each layer
- Design and compare bus, star, ring, and mesh topologies
- Configure IPv4 addressing, subnet masks, and perform subnetting calculations
- Distinguish key protocols: TCP, UDP, DNS, DHCP, ARP
- Select and terminate network cables (Cat5e, Cat6, fiber) and connectors

### üîí Module 4: Network Implementation and Security

**Topics covered:**

- Configure Cisco/managed switches and assign VLANs

- Apply ACLs, port security, and network access control (NAC)
- Set up and secure wireless networks (WPA2/WPA3, SSID management)
- Troubleshoot network connectivity with ping, traceroute, and packet capture
